# **Jomnay: Expense Tracking System**

**Jomnay** is an expense tracking system that helps users manage their daily spending and budget. It allows users to record expenses, view transaction history, and monitor their monthly allowance. The system also provides smart alerts and spending analysis to help users make better financial decisions.

## Key Features

* **Member Access:** Users can create accounts and log in to manage their data.
* **Monthly Allowance:** Calculates a monthly spending limit for every month.
* **Add Expense:** Record spending with date and category.
* **Expense History:** View all past transactions.
* **Smart Alerts:** Sends Telegram alerts when spending is near or over the limit.
* **File Exporter:** Export all expense data into a file.
* **Spending Trends:** Compare current and previous month spending.
* **Savings Goal Planner:** Tracks leftover budget and adds it to savings.

## Project Setup

### Prerequisites

* Python 3 or higher
* pip (Python package manager)

### Installation

1. **Clone the repository:**

   git clone https://github.com/pjeonggi/Jomnay.git

   cd Jomnay
2. **Install Dependencies:**

   pip install -r requirements.txt

### Run the Project

python main.py
